## States & Interactions

### State 1: Empty (Initial)
```
Input Panel:
- TextArea: empty, placeholder visible
- File Upload: button visible
- Run Scan: DISABLED (grayed out)

Results Panel:
- Empty (no tiles shown)
```

### State 2: Input Provided
```
Input Panel:
- TextArea: filled with text
- Run Scan: ENABLED (blue glow)

Results Panel:
- Still empty (waiting for click)
```

### State 3: Scanning (Loading)
```
Input Panel:
- All inputs: DISABLED
- Run Scan: shows "Scanning..."

Results Panel:
- Loading spinner
- Message: "Scanning with AI & Vision Models..."
```

### State 4: Results Received
```
Input Panel:
- Re-enabled after scan

Results Panel:
- All tiles appear with animation
- Scores visible
- Patterns listed
- Colors applied based on decision
```

### State 5: Error
```
Results Panel:
- Red error tile
- âŒ Error icon
- Error message in red text
- Allows retry
```
